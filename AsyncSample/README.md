Acync Sample (お天気アプリ)
====
非同期通信を行うAndroidアプリのサンプル

## Description
* Livedoorの天気API(Weather Hacks)を呼び東京、横浜、甲府、大阪、神戸の天気情報を表示するアプリ。
* Kotlinで実装
* 外部モジュールは使わないシンプルな構成

## Requirement
* Gradle vesion: 4.10.1
* Android Plugin Version: 3.3.2
* Compile Sdk Version : API28: Android 9.0(Pie)

## Usage
Android Studio の Run 'app'を実行
